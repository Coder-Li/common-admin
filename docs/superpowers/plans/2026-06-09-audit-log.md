# Audit Log Implementation Plan

> **For agentic workers:** REQUIRED: Use superpowers:subagent-driven-development (if subagents available) or superpowers:executing-plans to implement this plan. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Implement business audit logs for successful admin write operations, with sanitized snapshots, permission-protected read-only APIs, and a read-only admin table page.

**Architecture:** Add a local NestJS `AuditLogModule` backed by a Prisma `AuditLog` table. Business services write audit records through `AuditLogService.record(input, tx?)`, using the same transaction for normal database-only writes, while the admin frontend consumes the logs as a read-only CRUD-pattern resource at `/audit-logs`.

**Tech Stack:** NestJS, Prisma, PostgreSQL, class-validator, Swagger, Jest, React, Vite, TypeScript, TanStack Query, Axios, Vitest, React Testing Library, Tailwind CSS, lucide-react.

## Implementation Notes

- This feature records business write operations only. Do not add login, logout, refresh token, failed auth, validation failure, page view, file download, or analytics events.
- The permission code is `audit_log.read`, not `audit_logs.read`.
- The route path and feature folder are plural: `/audit-logs` and `apps/admin/src/features/audit-logs/`.
- `audit_log.read` should use `defaultRoles: []` because audit logs are sensitive. `super_admin` still receives it automatically.
- Audit log rows are append-only at the application API layer. Do not expose update or delete routes.
- All `before`, `after`, and `metadata` values must be sanitized before storage.
- Normal Prisma-only business writes must write the mutation and audit row in the same transaction.
- File upload has external storage side effects. Keep first-version compensation simple and follow the spec.

- [ ] **Step 4: Add sanitizer implementation**

Create `apps/api/src/audit-log/audit-log.sanitizer.ts`:

```ts
import { AUDIT_SENSITIVE_KEYS } from './audit-log.constants';

const REDACTED = '[REDACTED]';

export function sanitizeAuditPayload(value: unknown): unknown {
  if (Array.isArray(value)) {
    return value.map((item) => sanitizeAuditPayload(item));
  }

  if (!value || typeof value !== 'object') {
    return value;
  }

  return Object.fromEntries(
    Object.entries(value).map(([key, nestedValue]) => [
      key,
      AUDIT_SENSITIVE_KEYS.has(key.toLowerCase())
        ? REDACTED
        : sanitizeAuditPayload(nestedValue),
    ]),
  );
}
```

export function toAuditLogListItemResponse(
  log: AuditLog,
): AuditLogListItemResponseDto {
  return {
    id: log.id,
    ...(log.actorUserId ? { actorUserId: log.actorUserId } : {}),
    ...(log.actorEmail ? { actorEmail: log.actorEmail } : {}),

    ...(log.resourceId ? { resourceId: log.resourceId } : {}),
    ...(log.ipAddress ? { ipAddress: log.ipAddress } : {}),

export function toAuditLogResponse(log: AuditLog): AuditLogResponseDto {

import { Controller, Get, Param, Query } from '@nestjs/common';
import {
  ApiBearerAuth,
  ApiForbiddenResponse,
  ApiNotFoundResponse,
  ApiOkResponse,
  ApiTags,
} from '@nestjs/swagger';
import { Permissions } from '../auth/permissions.decorator';
import { AuditLogService } from './audit-log.service';
import { AuditLogListQueryDto } from './dto/audit-log.request';
import {
  AuditLogListResponseDto,
  AuditLogResponseDto,
} from './dto/audit-log.response';

@ApiTags('Audit Logs')
@ApiBearerAuth('access-token')
@Controller('audit-logs')
export class AuditLogController {
  constructor(private readonly auditLogService: AuditLogService) {}

  @ApiOkResponse({ type: AuditLogListResponseDto })
  @ApiForbiddenResponse({ description: 'Permission required' })
  @Permissions('audit_log.read')
  @Get()
  listAuditLogs(
    @Query() query: AuditLogListQueryDto,
  ): Promise<AuditLogListResponseDto> {
    return this.auditLogService.listAuditLogs(query);
  }

  @ApiOkResponse({ type: AuditLogResponseDto })
  @ApiForbiddenResponse({ description: 'Permission required' })
  @ApiNotFoundResponse({ description: 'Audit log not found' })
  @Permissions('audit_log.read')
  @Get(':id')
  getAuditLog(@Param('id') id: string): Promise<AuditLogResponseDto> {
    return this.auditLogService.findById(id);
  }
}

```ts
import type { Request } from 'express';
import type { JwtUserPayload } from '../user/user.types';
import type { AuditActor, AuditRequestMeta } from './audit-log.types';

type MinimalRequest = Pick<Request, 'ip' | 'headers'>;

export function buildAuditActor(user: JwtUserPayload): AuditActor {
  return {
    userId: user.sub,
    ...(user.email ? { email: user.email } : {}),
    ...(user.username ? { name: user.username } : {}),
  };
}

export function getAuditRequestMeta(request: MinimalRequest): AuditRequestMeta {
  const userAgent = request.headers['user-agent'];
  return {
    ...(request.ip ? { ipAddress: request.ip } : {}),
    ...(typeof userAgent === 'string' ? { userAgent } : {}),
  };
}
```
